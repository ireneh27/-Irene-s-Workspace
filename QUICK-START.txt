📝 STEP 3: ADD A NEW SLIDE (THE EASY WAY!)
───────────────────────────────────────────────────────────────────

A. Create your lesson HTML file
   - Use example-presentation.html as template
   - Customize with your content
   - Save as: my-lesson.html

B. Upload to slides folder
   - Go to slides/ folder
   - Upload my-lesson.html

C. Update slides-loader.js (ONLY 5 LINES!)
   - Open slides-loader.js
   - Find the slideFiles array
   - Add this:
   
   {
       filename: 'my-lesson.html',
       title: 'My Lesson Title',
       description: 'What students will learn...',
       tags: ['Grammar', 'Beginner', 'Fun']
   }
   
   - Commit changes

D. DONE! ✨
   Wait 2-3 minutes → Visit your website → New slide appears!

🎯 SIMPLE EXAMPLE
───────────────────────────────────────────────────────────────────

Let's say you want to add a "Colors" vocabulary lesson:

1. Create: colors-vocab.html
2. Upload to: slides/colors-vocab.html
3. Edit slides-loader.js, add:

{
    filename: 'colors-vocab.html',
    title: 'Vocabulary: Colors',
    description: 'Learn color words with pictures and games.',
    tags: ['Vocabulary', 'Beginner', 'Fun']
}

4. Commit → DONE! Your website shows the new lesson automatically!
