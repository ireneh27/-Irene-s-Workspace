🚀 QUICK SETUP (5 Steps)
─────────────────────────────────────────────────────────────────

STEP 1: CREATE IMAGES FOLDER
──────────────────────────────
1. Go to GitHub repository
2. Click "Add file" → "Create new file"
3. Type: images/README.md
4. Content: "Book cover images"
5. Commit

STEP 2: GET BOOK COVER IMAGES
──────────────────────────────
Option A: Download from publisher websites
Option B: Take photos of physical books
Option C: Screenshot from Amazon/Google Books

Save as JPG, 300-500px wide

STEP 3: UPLOAD IMAGES
──────────────────────────────
1. Go to images/ folder
2. Click "Upload files"
3. Upload your book cover images
4. Name them clearly (e.g., oxford-discover-1.jpg)
5. Commit

STEP 4: UPLOAD NEW FILES
──────────────────────────────
1. Delete old: content-loader.js, styles.css, index.html
2. Rename: index-updated.html → index.html
3. Upload: index.html, content-loader.js, styles.css
4. Commit

STEP 5: EDIT content-loader.js
──────────────────────────────
1. Open content-loader.js on GitHub
2. Click pencil icon (Edit)
3. Update the materials array (line ~20)
4. Add your books and image paths
5. Commit

💡 IMAGE TIPS
─────────────────────────────────────────────────────────────────

Good Image Names:
✅ oxford-discover-level-1.jpg
✅ cambridge-english-grade-2.jpg
✅ everybody-up-starter.jpg

Bad Image Names:
❌ Book Cover.jpg (spaces!)
❌ IMG_1234.jpg (not descriptive)
❌ Cover.JPG (use lowercase .jpg)

Image Size:
- Width: 300-500px
- Height: 400-700px
- File size: Under 500KB
- Format: JPG or PNG

⚠️ COMMON ISSUES
─────────────────────────────────────────────────────────────────

Image doesn't show:
→ Check filename matches exactly
→ Verify image is in images/ folder
→ Wait 5 minutes for GitHub to update

Links don't work:
→ Make sure URL starts with https://
→ Check for typos in publisherLink

Resources not clickable:
→ Make sure you uploaded the UPDATED styles.css
→ Clear browser cache (Ctrl + Shift + R)
